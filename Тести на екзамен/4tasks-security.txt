::Тестове завдання "Security"::
Що таке Thread Modeling?
{
=Процес, при якому потенційні загрози такі як структурні слабкості, можуть бути ідентифіковані, пронумеровані та надані пріоритети
~Передача функціонального навантаження на резервний компонент комп'ютерної системи або мережі в разі збою або порушення функціонування основного компонента тієї ж системи або мережі
~Система для спостереження за усіма процесами, конфігураційними одиницями, послугами, залежними одиницями та системою в цілому
}

::Тестове завдання "Security"::
Яке призначення Thread Modeling?
{
=Забезпечення замовників систематичним аналізом ймовірних вразливостей та векторів атак на систему
~Можливість здійснювати централізований контроль всіх параметрів інфраструктури 
~Моделювання та глибокий аналіз основних аспектів застосунку
}

::Тестове завдання "Security"::
Що таке STRIDE модель?
{
=Модель загроз для ідентифікації небезпек та потенційних загроз комп'ютерної безпеки 
~Модель для більш ефективного визначення пріоритетності роботи в області виявлення помилок
~Модель представлення взаємодії компонентів усієї системи між собою
}

Виберіть яку загрозу передбачає "Sproofing" із STRIDE моделі?
{
=Атака призначена для видавання себе за іншу особу, внаслідок злому з використанням Brute Force, фішингу або шкідливого програмного забезпечення
~Атака на цілісність системи з модифікацією частини аплікації (DLL наприклад) або вживленням (Injection) шкідливого коду у систему
~Деструктивна атака на систему, така як видалення всіх записів з бази даних або знищення логів для приховування атаки
~Атака призначена для перевантаження системи шляхом використання DDOS або іншої атаки, при якій сервіс або мережевий ресурс стає недоступний для користувачів
}
